# Assets Directory

This directory contains app assets like icons and images.

## Required Assets

To run the app properly, you'll need to add the following placeholder images:

- `icon.png` - App icon (1024x1024 px)
- `adaptive-icon.png` - Android adaptive icon (1024x1024 px)
- `splash.png` - Splash screen image (1284x2778 px)
- `favicon.png` - Web favicon (48x48 px)

For development, you can use simple colored squares as placeholders.

## Generating Assets

You can generate these using:
```bash
npx expo prebuild --clean
```

Or create simple placeholders manually.
